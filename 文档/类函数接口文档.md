### 推荐算法

**描述**

> 分析用户行为并返回推荐的服务

**输入参数**

| 参数名    | 必选 | 类型      | 说明               |
| --------- | ---- | --------- | ------------------ |
| `service` | `是` | `Service` | 用户购买的服务对象 |
| `count`   | `是` | `int`     | 推荐的服务个数     |

**返回参数**

| 参数名     | 必选 | 类型        | 说明                 |
| ---------- | ---- | ----------- | -------------------- |
| `services` | 是   | `Service[]` | 推荐给用户的服务顺序 |
